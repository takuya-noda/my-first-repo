# Linux基本コマンド集
# Linuxコマンドまとめ

Linuxの基本操作をカテゴリごとに整理した一覧です。随時追記・編集して使ってください。

---

## ファイル・ディレクトリ操作

| コマンド | 説明 |
|----------|------|
| `ls -al` | ファイル一覧を詳細表示（隠しファイル含む） |
| `cd ディレクトリ名` | ディレクトリ移動 |
| `pwd` | 現在のパスを表示 |
| `mkdir ディレクトリ名` | 新しいディレクトリを作成 |
| `touch ファイル名` | 空のファイルを作成 |
| `rm ファイル名` | ファイルを削除（`-r`で再帰、`-f`で強制） |
| `mv 元ファイル名 新ファイル名` | ファイルを移動／名前変更 |
| `cp 元ファイル名 コピー先` | ファイルをコピー |

---

## ファイルの中身を確認・編集

| コマンド | 説明 |
|----------|------|
| `cat ファイル名` | 中身を一括表示 |
| `less ファイル名` | ページ送りで閲覧（`q`で終了） |
| `head -n 10` | 先頭10行だけ表示 |
| `tail -n 10` | 最後の10行だけ表示 |
| `nano ファイル名` | 簡単なテキストエディタ |
| `vim ファイル名` | 高機能なエディタ（上級者向け） |

---

## 権限と所有者

| コマンド | 説明 |
|----------|------|
| `ls -l` | ファイルのパーミッションを表示 |
| `chmod +x ファイル名` | 実行権限を付ける |
| `chmod 755 ファイル名` | 数字指定でパーミッション設定 |
| `chown ユーザー:グループ ファイル名` | 所有者・グループ変更 |
| `sudo` | 管理者権限でコマンド実行 |

---

## 検索・確認

| コマンド | 説明 |
|----------|------|
| `find . -name "*.txt"` | カレント以下で.txtファイルを検索 |
| `grep 文字列 ファイル名` | ファイル内から文字列検索 |
| `which コマンド名` | 実行ファイルの場所を確認 |
| `history` | 実行したコマンド履歴を表示 |

---

##️ プロセスとシステム情報

| コマンド | 説明 |
|----------|------|
| `ps aux` | 実行中プロセスを表示 |
| `top` | CPU・メモリ状況のリアルタイム表示 |
| `kill PID` | プロセスを終了 |
| `free -h` | メモリの使用量を表示 |
| `df -h` | ディスクの使用量を表示 |
| `uptime` | システムの稼働時間を表示 |

---

## ネットワーク

| コマンド | 説明 |
|----------|------|
| `ping www.google.com` | 接続テスト |
| `ip a` | IPアドレスを確認（または `ifconfig`） |
| `ss -tuln` | 開いているポートの確認 |
| `netstat -tulnp` | 接続状況の確認（古いが現場で見る） |
| `curl https://example.com` | Webにリクエストを送る |

---

## ユーザー・グループ管理

| コマンド | 説明 |
|----------|------|
| `adduser ユーザー名` | 新しいユーザーを追加 |
| `passwd ユーザー名` | パスワードを設定／変更 |
| `id` | 自分のユーザーID・グループ確認 |
| `groups` | 所属しているグループ確認 |

---

## シャットダウン・再起動

| コマンド | 説明 |
|----------|------|
| `shutdown -h now` | 今すぐシャットダウン |
| `reboot` | 再起動 |
| `shutdown -r +5` | 5分後に再起動 |
