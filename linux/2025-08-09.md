# 2025/08/09 Linux学習記録

## 今日やったこと
- **ファイルとパーミッション操作**
  - `ls -l` で詳細表示（権限・所有者・グループ）
  - `chmod`（数値 / 記号モード）
  - `chown` / `chgrp` による所有者・グループ変更
- **プロセス管理**
  - `ps` / `top` でプロセス確認
  - `kill` / `kill -9` でプロセス終了
  - `nice` / `renice` でプロセス優先度の設定・変更
- **パッケージ管理（apt / dpkg）**
  - `apt update` / `apt upgrade` / `apt install` / `apt remove` / `apt purge`
  - `dpkg -l` / `dpkg -s` / `dpkg -L`
- **ユーザーとグループ管理**
  - `useradd` / `passwd` / `groupadd` / `usermod` / `groups`
  - `userdel` / `groupdel`
