## ポートフォリオ進行

就職活動に向けて、ポートフォリオリポジトリ「infra-portfolio-lite」を作成。  
MacのターミナルからParallels上のUbuntuを操作して構築練習を進行中。  
AIは補助として活用しつつ、実際の手動構築と検証を中心に進めている。

## GitHub開始日にやったこと
- GitHubにアカウントを作成
- 初めてのリポジトリ作成（my-first-repo）
- SSH鍵を生成してGitHubと接続
- hello.txt を作成してアップロード
- Macにクローンして  初成功

## GitHub反映メモ

GitHub上に変更を反映する手順（更新するたびに実行）：

- `nano README.md` で編集を開始する
- ファイルを保存して閉じる
- `git add README.md` で変更をステージに追加
- `git commit -m "コミットメッセージ"` で変更を記録
- `git push origin main` でGitHubに反映
