# my-first-repo
## GitHub開始日にやったこと
- GitHubにアカウントを作成
- 初めてのリポジトリ作成（my-first-repo）
- SSH鍵を生成してGitHubと接続
- hello.txt を作成してアップロード
- Macにクローンして  初成功

## GitHub反映メモ

GitHub上に変更を反映する手順（更新するたびに実行）：

- `nano README.md` で編集を開始する
- ファイルを保存して閉じる
- `git add README.md` で変更をステージに追加
- `git commit -m "コミットメッセージ"` で変更を記録
- `git push origin main` でGitHubに反映
